CI/CD Pipeline Automation using GitHub Actions and AWS EC2

Built an automated CI/CD pipeline for a Python Flask application using GitHub Actions. Implemented automated testing with PyTest, scheduled and manual workflow triggers, secure deployment to AWS EC2 using SSH keys and GitHub Secrets, and production-style service management using systemd. The pipeline automatically validates code, deploys updates to EC2, and restarts the application service without manual intervention.
